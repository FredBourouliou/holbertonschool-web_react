# Holberton School - Web React

Projets du curriculum **Web React** de Holberton School. Ce dépôt couvre la progression complète : des fondamentaux TypeScript jusqu'aux patterns avancés de React.

## Projets

| Projet | Description | Technologies |
|--------|-------------|--------------|
| [TypeScript](./TypeScript) | Interfaces, classes, génériques, namespaces, typage nominal | TypeScript, Webpack, Jest |
| [Webpack](./Webpack) | Configuration, loaders, code splitting, dev server | Webpack 5, jQuery, Lodash |
| [react_intro](./react_intro) | Création d'app React, composants de base, tests unitaires | React 19, Vite, Jest |
| [react_props](./react_props) | Props, composition de composants, rendu conditionnel | React 19, Vite, Jest |
| [react_component](./react_component) | Composants classe, lifecycle, HOC (Higher-Order Components) | React 19, Vite, Jest |

## Prérequis

- Node.js (v18+)
- npm

## Installation

```bash
# Cloner le dépôt
git clone https://github.com/FredBourouliou/holbertonschool-web_react.git

# Se placer dans un projet
cd holbertonschool-web_react/<projet>

# Installer les dépendances
npm install
```

## Lancement

Chaque projet contient ses propres scripts npm :

```bash
# Démarrer le serveur de développement (projets React)
npm run dev

# Lancer les tests
npm test

# Build de production
npm run build
```
