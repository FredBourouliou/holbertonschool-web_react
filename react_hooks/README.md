# React Hooks

This project covers the transition from class components to functional components using React Hooks.

## Learning Objectives

- Convert class components to functional components while preserving functionality
- Use React hooks: `useState`, `useContext`, `useEffect`, and custom hooks
- Optimize components with hooks
- Develop testing strategies for hooks
- Understand the reducer pattern before Redux

## Requirements

- Ubuntu 20.04 LTS, Node 20.x.x, npm 10.x.x
- Jest 29.7.0 (globally installed)
- React imported only in `main.jsx`
- All files end with a new line
