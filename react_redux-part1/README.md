# React Redux - Part 1

## Description

This project introduces Redux Toolkit and React-Redux into an existing React application. The focus is on restructuring the project into a scalable architecture with clear separation between components, pages, features (Redux slices), and shared utilities.

## Technologies

- React 19
- Redux Toolkit
- React-Redux
- Jest / React Testing Library
- Vite
- ESLint

## Requirements

- Ubuntu 20.04 LTS
- Node 20.x.x
- npm 10.x.x
- Jest 29.7.0 (global)

## Setup

```bash
cd task_0/dashboard
npm install
```

## Usage

```bash
npm run dev      # Development server
npm test         # Run tests
npm run lint     # Run linter
```

## Tasks

### 0. Update project structure

Reorganize the project into a Redux-ready architecture with `app/`, `features/`, `components/`, `pages/`, `hooks/`, and `utils/` directories. Install `@reduxjs/toolkit` and `react-redux`.
