# React Intro

## Description

Introduction to React with Vite. This project covers the basics of creating a React application, including component creation, JSX usage, utility functions, testing with Jest, and notifications management.

## Technologies

- React 18
- Vite
- Jest 29
- ESLint

## Tasks

| Task | Description |
|------|-------------|
| `task_0` | Initialize a React app with Vite, create the main App component and utility functions |
| `task_1` | Add tests for the App component and utility functions using Jest and Testing Library |
| `task_2` | Create a Notifications component with close button, priority-based notification list, and CSS styling |

## Setup

```bash
cd task_<number>/dashboard
npm install
```

## Usage

```bash
npm run dev
```

## Tests

```bash
npm test
```
